## 🚀 TO RUN THE SYSTEM

### 3 Simple Steps:

**1. Install Dependencies**
```bash
cd manufacturing-app
npm install
```

**2. Configure Database**
Create `.env` file with:
```env
MONGODB_URI=mongodb://localhost:27017/lac-bangles
PORT=5000
NODE_ENV=development
```

**3. Start Server**
```bash
npm start
```

**Open:** http://localhost:5000
